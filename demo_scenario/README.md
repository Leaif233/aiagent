# 演示场景：茶悦连锁奶茶店 — 智能设备运维支持系统

## 背景

"茶悦"是一家拥有 200+ 门店的连锁奶茶品牌。每家门店配备多种专业设备：
制冰机、封口机、果糖机、冷藏柜、净水器、收银一体机等。

## 痛点（没有本系统之前）

1. **知识分散**：设备手册散落在各门店，新员工找不到
2. **经验流失**：老店长离职后，积累的故障处理经验随之消失
3. **重复求助**：同样的问题反复打电话给总部技术部，效率低下
4. **响应慢**：总部技术人员有限，高峰期工单积压严重

## 本系统如何解决

- **技术文档入库**：将所有设备操作手册、维护指南结构化入库，经专家审核后可检索
- **工单经验沉淀**：每次故障处理后的工单（现象→根因→对策）自动提取、审核后入库
- **智能检索导航**：店员描述问题后，系统从已审核知识库中精准匹配，返回验证过的解决方案
- **零幻觉保障**：所有答案来自已审核内容，不会出现 AI 编造的错误指导
