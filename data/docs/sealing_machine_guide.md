# 封口机使用与故障排查指南

## 设备信息

- 型号：茶悦定制 SEAL-800
- 封口速度：3-5 秒/杯
- 适用杯径：90mm / 95mm / 98mm（更换模具切换）
- 封膜材质：PP/PET 复合膜，宽度 125mm
- 额定功率：0.5kW
- 工作温度范围：160°C - 200°C

## 一、操作流程

1. 接通电源，按【启动】键，加热指示灯亮起
2. 等待 2-3 分钟预热，温度达到设定值后提示音响起
3. 将饮品杯放入模具托盘，确认杯口平整
4. 下压手柄（或按下自动封口键），保持 2-3 秒
5. 听到"滴"声后松开，取出封好的杯子
6. 检查封口是否完整，无褶皱、无漏封

## 二、参数调节

| 饮品类型 | 建议温度 | 封口时间 |
|---------|---------|---------|
| 冷饮（冰杯） | 175°C | 3.0 秒 |
| 常温饮品 | 170°C | 2.5 秒 |
| 热饮 | 185°C | 3.5 秒 |

**注意**：杯壁有水珠时需擦干后再封口，否则容易封不牢。

## 三、常见故障排查

### 故障1：封口不牢，轻轻一戳就破
- **原因**：温度不足或封口时间过短
- **处理**：
  1. 检查设定温度是否匹配饮品类型（参考上表）
  2. 适当延长封口时间 0.5 秒
  3. 检查封膜是否受潮（受潮膜封口效果差，需更换）
  4. 检查加热板表面是否有残胶，用铜刷清理

### 故障2：封膜有褶皱或偏移
- **原因**：膜卷安装不正或张力不均
- **处理**：
  1. 重新安装膜卷，确保膜面平整居中
  2. 调节两侧张力旋钮，使膜面均匀绷紧
  3. 检查导膜辊是否有磨损，磨损严重需更换

### 故障3：加热不升温，温度显示不变
- **原因**：加热管损坏或温控器故障
- **处理**：
  1. 断电后用万用表测量加热管阻值（正常 30-60Ω）
  2. 检查温控探头是否脱落或接触不良
  3. 若加热管开路，需联系售后更换（型号：HT-SEAL-800）

### 故障4：封口后膜面烧焦发黄
- **原因**：温度过高或封口时间过长
- **处理**：
  1. 降低温度 5-10°C
  2. 缩短封口时间 0.5 秒
  3. 检查温控是否失灵（实际温度与显示温度偏差超过 10°C 需校准）
