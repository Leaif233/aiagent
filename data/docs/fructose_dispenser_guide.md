# 果糖机校准与日常维护手册

## 设备信息

- 型号：茶悦定制 SUGAR-16
- 出糖口数量：16 键（可分别设定不同出糖量）
- 果糖桶容量：5.5L
- 出糖精度：±0.5ml
- 额定功率：0.05kW
- 适用糖浆：果糖糖浆（55%浓度）、风味糖浆

## 一、基本操作

1. 将果糖桶安装到机器顶部卡槽，确认桶口密封垫完好
2. 接通电源，面板显示"Ready"
3. 按对应按键出糖，出糖量由预设决定
4. 果糖桶液位低于刻度线时及时更换

## 二、出糖量校准

茶悦标准甜度配方：

| 按键编号 | 对应产品 | 标准出糖量 |
|---------|---------|-----------|
| 1 | 无糖 | 0ml |
| 2 | 少糖（三分糖） | 8ml |
| 3 | 半糖 | 13ml |
| 4 | 标准糖 | 18ml |
| 5 | 全糖 | 22ml |
| 6-16 | 特调/季节限定 | 按配方卡设定 |

### 校准步骤

1. 长按【设置】键 3 秒进入校准模式
2. 按要校准的按键编号
3. 将量杯放在出糖口下方
4. 按【测试】键出糖，用电子秤称重（果糖密度约 1.3g/ml）
5. 若偏差超过 ±1ml，按【+】或【-】微调
6. 重复测试直到精度达标
7. 按【确认】保存，按【退出】返回

**建议**：每周一早班校准一次，换糖浆批次时必须重新校准。

## 三、常见故障排查

### 故障1：按键后不出糖
- **原因**：糖浆用尽、管路气泡或电磁阀卡死
- **处理**：
  1. 检查果糖桶是否见底，及时更换
  2. 连续按【测试】键 5-6 次排出管路气泡
  3. 拆下出糖嘴，用温水浸泡清洗（糖浆结晶堵塞）
  4. 若仍不出糖，检查电磁阀线圈是否通电

### 故障2：出糖量忽多忽少
- **原因**：管路内有气泡或齿轮泵磨损
- **处理**：
  1. 更换果糖桶后连按测试键排气
  2. 检查桶口密封垫是否老化漏气
  3. 重新校准出糖量
  4. 若校准后仍不稳定，齿轮泵可能磨损需更换

### 故障3：出糖口滴漏
- **原因**：出糖嘴残留糖浆或单向阀失效
- **处理**：
  1. 每日收工后用温水冲洗出糖嘴
  2. 检查出糖嘴内的硅胶单向阀是否变形
  3. 单向阀变形需更换（耗材编号：SV-SUGAR-16）
